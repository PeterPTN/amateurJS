Slice() method

The method() returns a new array based on a start and optional end index.

Here we use a start index of 2. Everything after the start index is included.

                    
const myArray = [1, 2, 3, 4, 5];

const threeNumbers = myArray.slice(2);

console.log(threeNumbers);

// Expected Output: Array [3, 4, 5]

Tangents

If the start index is 0 then the entire array is copied.


const allElements = myArray.slice(0);

console.log(allElements);

// Expected Output: 
Array [1, 2, 3, 4, 5]

This method does not mutate the calling array.

Using End Index

The second argument to slice is the ending parameter.

This slice(2, 4) call can be explained as:

Starting at index 2 and up until, but not including, index 4.


const myArray = ["cat", "dog", "goat", "chicken", "horse"];

const twoAnimals = myArray.slice(2, 4);
    
console.log(twoAnimals);
    
// Expected Output: Array ["goat", "chicken"]

Tangents

The ending index can greater than the length of the calling array. 


.slice(2, 10);

// Expected Output: Array
["cat", "dog", "goat", "chicken", "horse"]

It is also possible to use negative integers for the end index


.slice(0, -1);

// Expected Output: Array
["cat", "dog", "goat", "chicken"]

It can be understood as:

Include everything from index 2 but subtract 1 off.
