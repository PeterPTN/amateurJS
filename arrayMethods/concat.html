Concat() method

The concat() method is used to merge 2 or more arrays.

In this example we are simply concatenating a literal array with values 4 and 5 to our existing numbers array.

Our new array is stored in the variable fiveNumbers.

                    
const numbers = [1, 2, 3];

const fiveNumbers = numbers.concat([4, 5])

console.log(fiveNumbers);

Expected Output: Array [1, 2, 3, 4, 5]

Tangents

It's possible to add values that aren't in an array with concat().

For example this would produce the same result as in our example.

numbers.concat(4, 5)

Combining More Than 2 Arrays

Here we are referencing our variables array2 and array3 in our concat() method.

The result is stored within combinedArray.

There is no "real" limit on how many arguments you can pass to concat().


const array1 = [10, 11];
const array2 = [22, 23];
const array3 = [34, 35];

const combinedArray = array1.concat(array2, array3);

console.log(combinedArray);

// Expected Output: 
Array [10, 11, 22, 23, 34, 35];

Tangents

It is also possible to concatenate objects with concat().

For example:


const obj = {name: "Peter", id: 1};

console.log("Object Peter", obj);

// ["Object Peter, {name: "Peter, id: 1}]
