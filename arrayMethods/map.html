Map() method

The map() method creates a new array from the results of a function you provide.

Our new array will be stored with multipliedNumbers.

Within map's call () is our provided function.

Map() calls this provided function for each array element for a total of 5 times.

In our first iteration (first call of our provided function), number represents our first array element, 1.

Continuing this iteration, the function returns the value of number/1 * 2.

Our provided function is called again but now number represents the next array element, 2.

This process is repeated for every array element.

                    
const numbers = [1, 2, 3, 4, 5];

const multipliedNumbers = numbers.map(
    function(number) {
        return number * 2;
    });

console.log(multipliedNumbers);

//Expected output: Array [2, 4, 6, 8, 10];

Tangents

Methods are functions but simply relate to an object - in this case the Array object.

Essentially map() is a function which requires another function (that you provide) to work.

Typically you will see anonymous functions and other short-hand variations: 

.map((number) {
    return number * 2;
};

// or

.map(number => number * 2);

They both do the same thing.

If I console.log(numbers), it will remain the same. It doesn't mutate the existing array.

Meaning if we need to use the same array with map() but with a different function we can.

Another Use Case

Often data is stored within an array of objects. Our employees array is an example of that.

Each object in our array has 3 properties: firstName and lastName.

Like above, we .map (and loop) over our existing employees array.

Our parameter employee, now a local variable, represents each array item (as an object with 3 properties).

Our function returns a string using the employee local variable, using dot notation to access these values.

employee.firstName | employee.lastName

We are left with a new array stored within greetings to be used however we like.


const employees = [
    {
        firstName: "Bob",
        lastName: "Jane",
        id: 1
    },
    {
        firstName: "Lindsey",
        lastName: "Wagner",
        id: 2
    }
];

const greetings = employees.map((employee) => {
    return "Hello " + employee.firstName + " " + employee.lastName
});

console.log(greetings);

/* 
Expected Output: Array ["Hello Bob Jane", "Hello Lindsey Wagner"];
*/

Tangents

This use case is far more common in ReactJS where you could do something like this with an array of objects.


employees.map((employee) {
    return {
        <h1>{employee.id}<h1>
    }
})

This would return a series of <h1> tags with the value of employee.id instead of a new array.

The variable greetings can be chained with other methods like .join().


console.log(greetings.join());

// Expected Output: String
// Hello Bob Jane, Hello Lindsey Wagner
